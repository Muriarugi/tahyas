الطَّالِبَة
MAHASISWI


: اَلسَّلاَمُ عَلَيْكُمْ

نَدَى
: وَعَلَيْكُمُ السَّلاَمُ إِلْهَامٌ
: اِسْمِيْ نَدَى. أَنَا سُوْرِيَّةٌ.

نَدَى
Namaku Nada. Saya orang Syria  
: اِسْمِيْ إِلْهَامٌ. أَنَا سُعُوْدِيَّةٌ.

إِلْهَامٌ
Namaku Ilham. Saya orang Saudi  
: أَنَا طَالِبَةٌ فِيْ جَامِعَةِ دِمَشْقٍ. نَدَى
Saya mahasiswa di Universitas Damaskus  
: أَنَا طَالِبَةٌ فِيْ جَامِعَةِ أُمِّ الْقُرَى. إِلْهَامٌ
Saya mahasiswi di Universitas Ummul-Qura  
: فِيْ أَيِّ كُلِّيَّةٍ تَدْرُسِيْنَ.

نَدَى
Di Fakultas mana engkau belajar?  
: أَدْرُسُ فِيْ كُلِّيَّةِ التَّرْبِيَّةِ.

إِلْهَامٌ
Saya belajar di Fakultas Pendidikan  
: وَفِيْ أَيِّ كُلِّيَّةٍ تَدْرُسِيْنَ أَنْتِ ؟ إِلْهَامٌ
: أَدْرُسُ فِيْ كُلِّيَّةِ الطِّبِّ. نَدَى
Saya belajar di Fakultas Kedokteran  
: سَأَكُوْنُ مُدَرِّسَةٌ، إِنْ شَاءَ اللهُ. إِلْهَامٌ
Saya akan menjadi guru, insya Allah.  
: سَأَكُوْنُ طَبِيْبَةٌ، إِنْ شَاءَ اللهُ. نَدَى
Saya akan menjadi dokter, insya Allah.
